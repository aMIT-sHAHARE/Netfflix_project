# 🎬 Netflix Content Analysis

## 📖 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to uncover insights into content trends, genre distribution, ratings, and release patterns. The goal is to understand how Netflix’s content library has evolved over time using data-driven analysis and visualization.

## 🎯 Objectives

* Analyze distribution of Movies and TV Shows
* Identify popular genres and ratings
* Examine content growth over the years
* Explore country-wise content production
* Generate meaningful insights using visualizations

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Analysis Performed

* Data Cleaning and Preprocessing
* Handling Missing Values
* Exploratory Data Analysis (EDA)
* Trend Analysis
* Data Visualization

## 📁 Dataset Features

The dataset includes the following attributes:

* Title
* Type (Movie / TV Show)
* Director & Cast
* Country
* Release Year
* Rating
* Duration
* Genre

## 📈 Key Insights

* Movies form the majority of Netflix content
* Significant content growth observed after 2015
* Certain genres dominate the platform
* Netflix content is produced across multiple countries
